This is Spacewar!

A frenetic multiplayer space shooter.
Shoot other ships before they shoot you.

Player controls

  • ↑ to thrust forward.
  • ← and → to rotate the ship.
  • A to shoot (hold for stronger bullets).
  • Z to use bonus (can only hold one).

How

  • HTML5 canvas and WebSocket
  • Node.js with Socket.IO
  • GNU Emacs
  • Source hosted on Github
